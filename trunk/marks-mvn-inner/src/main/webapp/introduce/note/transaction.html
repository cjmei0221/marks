事务提醒

1，事务提醒 ，特殊日子可设置每年提醒，普通事务提醒，默认明天；
2，提醒时间是9点提醒
3，可设置提前提醒

事务提醒表 (TB_NOTE_TRANSACTION)

序号 字段 名称 数据类型 备注
1IDIDvarchar2主键
2TRANTYPE事务类型number0：普通 1：特殊日子
3TRANDATE特殊日期TIMESTAMP无
4ISREPEAT是否重复number0：不重复 1：重复，默认不重复
5TRANCONTENT提醒内容varchar2无
6AHEADDAYS提前天数number无
7TRANTIME提醒时间varchar2无
8ISAHEAD是否提前提醒number0：不提前 1：提前
9CREATETIME创建时间TIMESTAMP无
10UPDATETIME更新时间TIMESTAMP无
11CREATOR创建者varchar2无

序号 字段 名称 数据类型 备注
1idIDString主键

3tranDate特殊日期Date无
4isRepeat是否重复int0：不重复 1：重复，默认不重复
5tranContent提醒内容String无
6aheadDays提前天数int无
7tranTime提醒时间String无
8isAhead是否提前提醒int0：不提前 1：提前

10updatetime更新时间Date无
11creator创建者String无
