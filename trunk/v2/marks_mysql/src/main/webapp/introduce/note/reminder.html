事务提醒

1，事务提醒 ，特殊日子可设置每年提醒，普通事务提醒，默认明天；
2，提醒时间是9点提醒
3，可设置提前提醒

事务提醒表 (TB_NOTE_REMINDER)

序号 字段 名称 数据类型 备注
1 ID ID varchar2 主键
2 REMIND_TYPE 事务类型 number 0：普通 1：特殊日子
3 REMIND_DATE 特殊日期 varchar2 无
4 IS_REPEAT 是否重复 number 0：不重复 1：重复，默认不重复
5 REMIND_CONTENT 提醒内容 varchar2 无
6 BEFORE_DAYS 提前天数 number 无
7 REMIND_TIME 提醒时间 varchar2 无
8 IS_BEFORE 是否提前提醒 number 0：不提前 1：提前
9 CREATETIME 创建时间 TIMESTAMP 无
10 UPDATETIME 更新时间 TIMESTAMP 无
11 CREATOR 创建者 varchar2 无
12 CALENDAR_TYPE 日历类型 number 0：公历 1：农历

实体类 (Reminder)

序号 字段 名称 数据类型 备注
1 id ID String 主键
2 remind_type 事务类型 int 0：普通 1：特殊日子
3 remind_date 特殊日期 String 无
4 is_repeat 是否重复 int 0：不重复 1：重复，默认不重复
5 remind_content 提醒内容 String 无
6 before_days 提前天数 int 无
7 remind_time 提醒时间 String 无
8 is_before 是否提前提醒 int 0：不提前 1：提前
9 createtime 创建时间 Date 无
10 updatetime 更新时间 Date 无
11 creator 创建者 String 无
12 calendar_type 日历类型 int 0：公历 1：农历

控制层 (ReminderController)

序号 URL 方法 备注
1 /reminder/findReminderById findReminderById 查询事务提醒
2 /reminder/save saveReminder 保存事务提醒
3 /reminder/update updateReminder 更新事务提醒
4 /reminder/delete deleteReminderById 删除事务提醒
5 /reminder/findAllReminder findAllReminder 查询全部事务提醒
6 /reminder/deleteIds deleteReminder 删除多个事务提醒
7 /reminder/list list 列表查询事务提醒
